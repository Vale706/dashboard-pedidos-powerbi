<h1>📊 Análisis de Pedidos y Comportamiento de Clientes — Power BI</h1>

<h2>📌 Descripción</h2>
<p>
Este proyecto consiste en el desarrollo de un dashboard interactivo en <strong>Power BI</strong> 
orientado al análisis de pedidos y comportamiento de clientes.
</p>

<p>
A través de visualizaciones dinámicas se exploran patrones de compra según género, 
materiales preferidos, productos más vendidos y frecuencia de compras por cliente, 
permitiendo identificar tendencias relevantes y generar insights para la toma de decisiones.
</p>

<h2>🎯 Objetivos</h2>
<ul>
  <li>Analizar el comportamiento de compra de los clientes</li>
  <li>Identificar productos y materiales más demandados</li>
  <li>Explorar diferencias de consumo según género</li>
  <li>Detectar clientes con mayor volumen de pedidos</li>
  <li>Generar visualizaciones claras para el análisis de datos</li>
</ul>

<h2>📊 Secciones del Dashboard</h2>
<ul>
  <li>📈 Género que más compra</li>
  <li>🧵 Materiales más elegidos</li>
  <li>👥 Producto más comprado por género</li>
  <li>🛒 Compras por cliente</li>
  <li>📦 Métrica total de pedidos</li>
</ul>

<h2>🛠️ Herramientas Utilizadas</h2>
<ul>
  <li>Power BI Desktop</li>
  <li>Power Query (transformación de datos)</li>
  <li>Modelado de datos</li>
  <li>Visualización interactiva</li>
</ul>
